Welcome to your new blog powered by [Hexo](https://hexo.io/) and the **hanli** theme!

## Quick Start

### Create a new post

```bash
hexo new "My New Post"
```

### Run the server

```bash
hexo server
```

### Generate static files

```bash
hexo generate
```

### Deploy to remote sites

```bash
hexo deploy
```

More info: [Hexo Documentation](https://hexo.io/docs/)
